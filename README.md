# GameDrop

Track game deals across Steam, PlayStation, Xbox, and Nintendo. Never miss a sale.

## Features

- 🔍 Search games across all major platforms
- 💰 Track prices and get deal alerts
- 📊 Price history tracking
- 📱 Works on desktop and mobile

## Tech Stack

- Next.js 15
- Tailwind CSS
- CheapShark API (Steam)
- PSPrices API (PlayStation)

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000
